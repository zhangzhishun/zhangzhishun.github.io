一、前提条件

需要连接互联网，然后执行sudo apt-get update更新软件包

四、测试redis服务（可选）

步骤三的操作完成后，重启redis服务

$ redis-server       # 启动服务
$ redis-cli          # 启动客户端
$ redis-cli shutdown # 关闭服务

如果出现以上命令无法关闭redis-server的情况下解决办法如下：

使用以下命令启动重启和关闭

$ /etc/init.d/redis-server stop # 停止
$ /etc/init.d/redis-server start # 启动
$ /etc/init.d/redis-server restart # 重启
